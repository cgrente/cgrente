# 💫 About Me:
👋 Hi, I’m Cliff — a Software Engineer with 10+ years of experience designing and scaling backend and full-stack systems, distributed architectures,
and cloud deployments. Strong focus on API design, production reliability, and CI/CD automation across global, customer-facing
platforms.<br>

I mostly work on:
- Designing scalable APIs and distributed services
- Building backend systems with clean architecture
- Shipping production-ready infrastructure with CI/CD and cloud tooling

Currently:
- Working on ERP-style systems

I enjoy projects where architecture, data, and long-term maintainability matter,
and where learning and improving systems is part of the work.
